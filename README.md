1. Complete DevOps path first
2. Deep dive into [DevSecOps](./DevSecOps/) - All security topics
3. Practice [Security Labs](./Hands-On/security-labs.md) and [SecOps Projects](./Projects/secops-projects.md)
4. Prepare with [DevSecOps Interview](./Interview/devsecops.md)

### Path 3: AI SecOps Specialist (18-24 months)
1. Master DevSecOps fundamentals
2. Study [AI-SecOps](./AI-SecOps/) - LLM security, prompt injection, model security
3. Complete [AI SecOps Labs](./Hands-On/ai-secops-labs.md) and [AI Security Projects](./Projects/ai-security-projects.md)
4. Interview prep: [AI Security Interview](./Interview/ai-security.md)

### Path 4: AIOps Engineer (18-24 months)
1. Strong DevOps + observability background
2. Learn [AIOps](./AIOps/) - Anomaly detection, automated remediation
3. Practice [AIOps Labs](./Hands-On/aiops-labs.md) and [AIOps Projects](./Projects/aiops-projects.md)
4. Interview prep: [AIOps Interview](./Interview/aiops.md)

### Path 5: Platform Engineer (24-30 months)
1. Senior DevOps experience required
2. Study [Platform Engineering](./Platform-Engineering/) - IDPs, Backstage, golden paths
3. Complete [Platform Engineering Projects](./Projects/platform-engineering-projects.md)
